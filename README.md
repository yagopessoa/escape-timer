# Escape Timer

Aplicação simples de timer para escape game, com função de penalidade (-30s) criado utilizando ReactJS.

## Requisitos

Ter Node.Js e Npm instalado.

## Rodar o projeto

Entre no diretório do projeto e rode os comandos:

`npm install`

`npm start`
